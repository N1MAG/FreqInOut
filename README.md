# FreqInOut

See docs/Installation.md for installation and usage. Quick Debian/Ubuntu setup:

- Install base Python tooling (no PPAs): `sudo apt-get install python3 python3-venv python3-pip`
- Create and activate a virtualenv: `python3 -m venv venv && source venv/bin/activate`
- Install Python deps: `pip install -r requirements.txt`
- QtWebEngine prereqs for the Stations Map: `sudo apt-get install libxcb-cursor0 libxcb-xinerama0`
